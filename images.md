# images/

Place your image assets in this folder.

Recommended naming convention:
- hero-bg.jpg        → Hero background (if added later)
- og-image.jpg       → Open Graph / social share preview (1200×630px)
- favicon.png        → Site favicon (32×32px or 64×64px)
- logo.svg           → Vector logo (if you have one)

Reference images in HTML like:
  <img src="images/og-image.jpg" alt="Gridwright Studio">

Or in CSS like:
  background-image: url('images/hero-bg.jpg');
